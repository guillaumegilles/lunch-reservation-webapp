# Implementation Plan: Modern Website Styling

**Branch**: `002-modern-website-styling` | **Date**: 2025-06-03 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `/specs/002-modern-website-styling/spec.md`

## Summary

Replace the existing glassmorphism / gradient-heavy `style.css` with a flat minimal design system delivered as a single `custom.css` file. The redesign retains and refines the warm palette (berry, plum, apricot, sun), introduces a sticky coloured navbar, moves all design values into CSS custom properties, and ensures WCAG AA compliance across all six pages — with zero back-end changes and no build tooling.

## Technical Context

**Language/Version**: Python 3.12 (Vercel runtime) / Django 5.2.13

**Primary Dependencies**: Bootstrap 5.3.2 (CDN), Google Fonts (Archivo Black + DM Sans, CDN)

**Storage**: N/A — purely front-end CSS change; no database schema changes

**Testing**: pytest / Django test client (`reservations/tests/`) for regression; manual visual QA + browser DevTools contrast checker

**Target Platform**: Desktop browsers + mobile (≥ 375 px viewport), Vercel serverless deployment

**Project Type**: Internal web application (Django, server-side rendered)

**Performance Goals**: Hover/focus state feedback ≤ 200 ms (CSS `transition` only); no new render-blocking resources beyond existing Bootstrap + Google Fonts CDN

**Constraints**: Single `custom.css` file; no CSS preprocessor, no build step, no npm; no back-end changes; no dark mode; French UI copy untouched

**Scale/Scope**: 6 pages (index, login, register, dashboard, calendar, admin); 1 shared `base.html`; ~1 static file replaced + 2 templates modified (navbar added to `base.html`, inline `<style>` removed from `calendar.html`)

**Structure Decision**: Single-app Django project. All CSS in `reservations/static/custom.css`; all templates in `reservations/templates/`. Only `base.html` and `calendar.html` require markup edits; all other templates inherit the design system passively through Bootstrap class overrides.

## Complexity Tracking
